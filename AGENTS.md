## Editorial publishing rules

- Draft posts may contain unresolved content gaps.
- Published posts must not contain any raw placeholders:
  - [待补数据]
  - [图表占位]
  - [待补参考资料]
- If a post still needs real data, real charts, or verified references, keep it as draft.
- Never fabricate data, charts, citations, or source URLs just to make a post look complete.
- `status` and `draft` must be consistent.
- Run the content validation script before considering a content task done.
