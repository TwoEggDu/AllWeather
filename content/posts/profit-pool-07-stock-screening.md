## 一、利润池里 3-5 个标的，怎么挑？

第三刀帮你找出"光模块是利润池"。但光模块这个利润池里，A 股至少有 5 家公司：中际旭创、新易盛、天孚通信、太辰光、华工科技。

5 家都做光模块，都受益。但 5 个不能都买——资金有限、注意力有限，**相关性高的话买 5 家不如买 1 家加杠杆**。

那挑哪个？凭直觉买"龙头"？那为什么不买"二线弹性"？

我用三道筛把这个决策拆开：

1. **业务纯度**（主营占比）
2. **业绩档位**（5 档分级的输出）
3. **回撤归因**（三种来源的拆解）

三道筛跑完，5 家会被分到不同的仓位类别——有的进底仓、有的进进攻、有的进主题、有的不进。

## 二、第一道筛：业务纯度

### 怎么算

打开年报，看主营业务收入构成。**目标业务占比越高，纯度越高**：

- 高纯度（>80%）：涨跌主要由该赛道驱动
- 中纯度（40-80%）：该赛道是"重要但非全部"
- 低纯度（<40%）：该赛道只是"催化剂"，公司主体业务在别处

### 纯度的含义

- **高纯度 = 高弹性 + 高波动**：赛道好它弹得大，赛道差它跌得也猛
- **低纯度 = 低弹性 + 低波动**：赛道好它弹有限，赛道差它有别的支撑

### 实例：AI 算力链相关标的纯度

| 标的 | 主营 | 算力链占比 | 纯度 |
|---|---|---|---|
| 中际旭创 | 光模块 | >95% | 高 |
| 新易盛 | 光模块 | >90% | 高 |
| 天孚通信 | 光器件 | >85% | 高 |
| 沪电股份 | PCB | 50-60%（汽车 + 算力） | 中 |
| 工业富联 | EMS 代工 | ~30%（大部分消费电子） | 低 |

### 纯度对仓位的含义

- 高纯度 → 适合**主题仓和进攻仓**（赛道驱动力大）
- 中纯度 → 适合**底仓**（兼顾弹性和稳定）
- 低纯度 → 适合 **β 仓或价值底仓**（主要看公司本身）

## 三、第二道筛：业绩档位

这是第四刀的输出，直接复用 5 档分级：

- ①-② 档：主题股，进**主题仓**
- ③ 档：**进攻仓**（边际改善期）
- ④ 档：**核心进攻或底仓**
- ⑤ 档：**底仓**（稳态）

### 实例：AI 算力链各标的的档位

| 标的 | 档位 | 备注 |
|---|---|---|
| 中际旭创 | ④-⑤ | 已开始进入现金流阶段 |
| 新易盛 | ④ | 利润快速释放中 |
| 天孚通信 | ④ | 高毛利稳态 |
| 沪电股份 | ④ | 高端 PCB 业绩兑现 |
| 工业富联 | ⑤ | 大体量稳态 |
| 寒武纪 | ②-③ | 仍在亏损但收入起量 |
| 海光信息 | ③-④ | 利润释放期 |

## 四、第三道筛：回撤归因

这是第五刀的输出，直接复用：

- **β 主导** → 仓位分散应对
- **α 主导** → 财报季减仓
- **估值压缩主导** → 估值高位减仓
- **三因素叠加** → 高警惕，**不进底仓**

## 五、三筛合一：仓位分配矩阵

把三筛结果合在一起：

| 标的 | 业务纯度 | 档位 | 主导风险 | 仓位分配 |
|---|---|---|---|---|
| 中际旭创 | 高 | ④-⑤ | β | **底仓** |
| 工业富联 | 低 | ⑤ | β | **β 底仓**（低波动） |
| 新易盛 | 高 | ④ | β + 估值 | **进攻仓** |
| 天孚通信 | 高 | ④ | 估值压缩 | **稳健仓** |
| 沪电股份 | 中 | ④ | β | **底仓** |
| 海光信息 | 高 | ③-④ | 三因素 | **进攻仓**（有上限） |
| 寒武纪 | 高 | ②-③ | 三因素 | **主题仓**（有上限） |

**仓位类别的总体逻辑**：

- **底仓**：高/中纯度 + ④-⑤ 档 + β 主导风险 = **持仓信心高，适合长期持有**
- **进攻仓**：高纯度 + ③-④ 档 = **有弹性，吃边际改善逻辑**
- **主题仓**：高纯度 + ①② 档 = **高弹性高风险，仓位严控**
- **β 仓**：低纯度 + ⑤ 档 = **跟随大盘，稳但弹性小**

## 六、几个判断细节

### 细节 1：同一个利润池，不要全买

即使三筛过完都是底仓级标的，也不要全买。

- 同一利润池的标的相关性极高（中际旭创 vs 新易盛 vs 天孚通信）
- **持有 1-2 家足以**，多买不增加分散度

### 细节 2：跨利润池配置才是真正的分散

真正分散的是：1 家光模块 + 1 家 PCB + 1 家液冷，**而不是 3 家光模块**。

### 细节 3：档位低 + 纯度高的标的，仓位上限严控

寒武纪、鼎智科技这类标的，即使逻辑再硬，**单股仓位 ≤ 5%**。任何一次黑天鹅（业绩证伪、政策变化）都可能 -50%，整个组合不能被这种波动主导。

### 细节 4：财报季前后的仓位调整

- **档位 ②-③**：财报季前减仓 1/3（业绩证伪风险）
- **档位 ④**：财报季前持有（业绩超预期更可能）
- **档位 ⑤**：不调整（业绩波动小）

## 七、典型错配

### 错配 1：把 ②-③ 档的高纯度标的放底仓

"我看好这个赛道" → 把寒武纪放底仓 50% → 一次业绩证伪，**整个组合 -25%**。

**预防**：底仓只放 ④-⑤ 档。底仓的资格不取决于"我多看好"，取决于"它能不能扛住一次业绩证伪"。

### 错配 2：把 ⑤ 档的低纯度标的当主题仓

"工业富联是 AI 受益股" → 当主题仓追涨 → **弹性远低于纯标的**。

**预防**：主题仓只放 高纯度 + 有上行催化剂的标的。低纯度的票顶多是 β 底仓，不是主题。

### 错配 3：同一利润池过度集中

全仓买光模块（中际 + 新易 + 天孚）→ 业绩超预期都赢，但只要利润池逻辑松动（CPO 加速、海外 capex 拐点），**全部一起跌**。

**预防**：跨利润池配置才是真分散。

## 八、收尾

完成三道筛，你应该有：

1. 关注利润池里每个标的的**三筛画像**
2. 每只标的的**仓位类别**（底仓 / 进攻 / 主题 / β）
3. 每只标的的**仓位上限**和**调整时点**

但这只是"个股层面"完成。还差最后一步——**怎么把这堆个股+仓位类别组装成一个有机的组合，让它的整体回撤、相关性、对冲机制都符合你的目标**？

下一篇讲 **第七刀：从个股到组合**——给你一个 β 仓 + α 仓 + 主题仓的三层结构，加上事件日历和节奏管理。这一刀切完，整套方法论就走到了实盘可用的层面。
