## 这篇文章要回答的问题是什么

读完前面 25 篇关于五年规划、资源专题、地缘政治的文章后，你已经有了系统认知。但当你需要做实际决策时，仍然会遇到一个具体问题：

**当前这个时点，应该重点配置什么资源？回避什么？**

这篇文章把所有专题文章的核心结论压缩成一张**决策矩阵**——以"五年规划阶段"为横轴，"资源价格周期"为纵轴，给出每个组合下的资源配置建议。

这是给"未来的你"准备的工具。当你三年后想做决策时，不需要重读所有文章，直接查这张矩阵。

## 矩阵的两个维度

### 横轴：五年规划阶段

每个五年规划期间可以分为四个阶段：

| 阶段 | 时间窗口 | 资源相关特征 |
|------|---------|-----------|
| **起草定调期** | 启动年前一年 | 新方向预期升温，相关资源开始定价 |
| **落地启动期** | 启动年 | 政策细节落地，配套投资开始 |
| **中期执行期** | 启动后 2–3 年 | 实物需求兑现，价格走势分化 |
| **收官评估期** | 最后一年 | 完成度评估，下一周期方向开始酝酿 |

**当前位置**：2026 年 4 月，十五五处于"落地启动期"。

### 纵轴：资源价格周期

不同资源有自己的周期阶段：

| 阶段 | 价格特征 | 投资策略 |
|------|---------|---------|
| **底部** | 大幅下跌后企稳，库存高、产能过剩 | 分批建仓 |
| **复苏** | 价格小幅上涨，库存开始去化 | 加仓 |
| **景气** | 价格持续上涨，需求强劲，产能紧张 | 持有，警惕过热 |
| **顶部** | 价格创新高，新增产能开始释放 | 减仓 |
| **下行** | 价格下跌，产能集中释放 | 回避 |

## 矩阵主体：八种关键资源 × 当前位置

下表把每种关键资源的当前周期阶段，和与十五五的关系，放在一起：

| 资源 | 当前价格周期 | 与十五五关系 | 配置建议 |
|------|-----------|------------|---------|
| **铜** | 复苏到景气（缺口扩大）| 电气化必需 | **超配** |
| **铀** | 景气期（80–100 美元/磅）| 核电扩张直接受益 | **超配** |
| **黄金** | 景气期（高位，央行持续买入）| 去美元化叙事延续 | **标配偏超配** |
| **铂金** | 复苏期（供给约束 + 氢能预期）| 长期受益 | **小配** |
| **白银** | 复苏期（光伏需求支撑）| 新能源需求 | **小配** |
| **锂** | 底部到复苏（产能过剩出清中）| 新能源车需求 | **机会性建仓** |
| **镍** | 底部（印尼供给压力）| 电池需求 | **观望** |
| **石油** | 中性（70–85 美元）| 长期承压（电气化）| **标配** |
| **铁矿石** | 顶部到下行（中国地产需求弱化）| 需求结构性下降 | **低配或回避** |
| **煤炭** | 高位（中国仍依赖）| 长期承压但短期需求稳 | **标配偏低** |
| **稀土** | 复苏期（出口管制后）| 中国战略地位 | **小配** |
| **大豆/粮食** | 中性 | 粮食安全战略 | **小配防御** |

**当前（2026 年 4 月）的总体方向**：
- **超配**：铜、铀、黄金（与十五五碳排放双控、能源转型、去美元化叙事高度匹配）
- **标配**：石油、煤炭、其他基础金属
- **机会性配置**：锂、镍（在底部分批建仓，等待产能出清）
- **低配/回避**：铁矿石（中国地产需求结构性下降）

## 不同规划阶段下的资源配置思路

### 阶段一：起草定调期（如 2025 年）

**特点**：
- 新方向预期升温
- 市场开始定价新主题
- 但具体细节未明，分歧大

**策略**：
- 加配预期受益的"新方向"资源（如十五五前期市场对核电、AI、新能源的提前布局）
- 减配预期被边缘化的"旧方向"资源
- 保持现金/债券作为应变备用金

### 阶段二：落地启动期（2026 年）

**特点**：
- 《纲要》通过，具体目标明确
- 配套政策陆续出台
- 投资者预期被验证或调整

**策略**：
- 增加约束性指标对应的资源（如十五五的碳排放双控 → 核电、清洁能源）
- 谨慎对待"叙事强但执行不确定"的方向
- 对长期看好的资源做基础配置

### 阶段三：中期执行期（如 2027–2028 年）

**特点**：
- 政策落地兑现，实物需求开始释放
- 部分行业出现产能过剩
- 价格走势分化

**策略**：
- 区分"真受益方"和"伪受益方"——前者业绩兑现，后者炒概念后回归基本面
- 关注产能过剩信号，避免追入"红海"行业
- 增加现金或防御性资产权重，为下一阶段调整准备

### 阶段四：收官评估期（如 2030 年）

**特点**：
- 规划目标完成度评估
- 下一规划方向开始酝酿
- 市场在"老主题"和"新主题"之间切换

**策略**：
- 适度减仓"已经兑现"的方向（避免高位接盘）
- 关注"下一规划"可能强调的新主题（提前布局）
- 增加防御性资产，迎接周期切换

## 价格周期 × 计划阶段 的二维矩阵

把两个维度交叉，得到一张更细的决策表：

| | 起草定调期 | 落地启动期 | 中期执行期 | 收官评估期 |
|---|---------|---------|---------|---------|
| **底部** | 分批建仓（高赔率）| 继续建仓 | 持有等待 | 持有 |
| **复苏** | 加仓（趋势 + 政策双共振）| 加仓 | 持有 | 减仓部分 |
| **景气** | 持有（享受趋势）| 持有 | 减仓部分（预警过热）| 大幅减仓 |
| **顶部** | 减仓（政策可能转向）| 大幅减仓 | 回避 | 回避 |
| **下行** | 回避 | 回避（等待出清完成）| 谨慎机会性建仓 | 分批建仓 |

**使用方法**：
1. 确定当前所处的规划阶段（横轴）
2. 判断目标资源的价格周期（纵轴）
3. 找到交叉点的建议
4. 结合自己的风险偏好微调

## 个案：当前（2026 年 4 月）的应用示例

**示例一：铜**
- 价格周期：**复苏期到景气期**（库存去化中，长期供给缺口）
- 规划阶段：**落地启动期**（十五五碳排放双控带来电气化需求）
- 矩阵建议：**加仓**（趋势 + 政策双共振）

**示例二：铀**
- 价格周期：**景气期**（80–100 美元/磅，长期上行通道）
- 规划阶段：**落地启动期**（十五五核电核准节奏创新高）
- 矩阵建议：**持有**（享受趋势，但留意短期回调）

**示例三：锂**
- 价格周期：**底部到复苏期**（碳酸锂从 60 万跌至 8 万后企稳）
- 规划阶段：**落地启动期**（新能源车产业延续，但产能过剩）
- 矩阵建议：**继续建仓**（底部分批，长期看好需求）

**示例四：铁矿石**
- 价格周期：**顶部到下行**（中国房地产长期需求下降 + 西芒杜投产）
- 规划阶段：**落地启动期**（十五五对房地产明显降温）
- 矩阵建议：**减仓或回避**

## 结合全天候组合的实操

对于一个全天候组合，资源配置占比建议如下（在十五五落地启动期）：

| 资源类别 | 全天候组合权重 |
|---------|--------------|
| 黄金 | 10–12% |
| 铜（实物 ETF + 矿业股）| 3–5% |
| 铀（实物 + 矿业股 + 核电运营）| 3–5% |
| 石油/能源 | 3–5% |
| 锂/镍/钴/稀土等关键矿产 | 2–4% |

剩余 63–77% 配置股票、债券、现金。

## 反例 / 风险是什么

**第一，"周期阶段"判断本身有不确定性。** 你以为的"底部"可能不是底部。任何分阶段配置都需要分批，不要一次性满仓。

**第二，规划阶段和市场反应不同步。** 落地启动期可能"利好出尽是利空"——市场提前定价，《纲要》通过反而引发回调。

**第三，资源价格的"跨期波动"巨大。** 一个资源在景气期可能涨 100%，在下行期可能跌 70%。不要因为"长期看好"就忽视短期波动。

**第四，地缘事件可能彻底打乱矩阵。** 任何重大冲突（台海、中东、俄乌）都可能让"理性"的矩阵失效。要有应急方案。

**第五，矩阵是简化模型，不是精确预测。** 它给方向，不给具体的买入卖出价格。结合自己的判断使用。
