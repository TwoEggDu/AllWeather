## 这篇文章是什么

很多投资者的"组合管理"只是看券商账户的总市值。但完整的组合追踪需要：

- 跨账户合并（A 股、港股、美股、基金、银行理财、房产等）
- 按资产类别分类（不只是按账户）
- 当前权重 vs 目标权重对比
- 再平衡触发判断
- 收益归因分析

这篇文章给出可直接复制使用的模板。

### 关键计算

**总金融资产**：所有"金融资产"市值合计（不含自住房）

**权重**：单项市值 / 总金融资产

**偏差**：当前权重 - 目标权重

**操作建议**：
- 偏差 > +3%：减仓
- 偏差 -3% 到 +3%：维持
- 偏差 < -3%：加仓

## 模板 C：再平衡触发表

每月检查，决定是否需要操作。

```markdown
# YYYY-MM 再平衡检查

## 触发条件检查

### 个别资产偏差
- A 股权重偏差：[+/- X%]  [ ] 触发再平衡（>3%）
- 港股权重偏差：[+/- X%]  [ ] 触发再平衡
- 美股权重偏差：[+/- X%]  [ ] 触发再平衡
- 国债权重偏差：[+/- X%]  [ ] 触发再平衡
- 黄金权重偏差：[+/- X%]  [ ] 触发再平衡
- 商品权重偏差：[+/- X%]  [ ] 触发再平衡

### 整体触发
- 任意单一资产偏差 > 5%：必须再平衡
- 多个资产偏差 > 3%：建议再平衡
- 所有偏差 < 3%：不需要操作

## 再平衡操作清单

如果触发，执行：

| 操作 | 标的 | 数量 | 预计金额 | 实际执行 |
|------|------|------|--------|--------|
| 卖出 | | | | |
| 卖出 | | | | |
| 买入 | | | | |
| 买入 | | | | |

## 操作后检查

执行后再计算所有权重，确认偏差都回到 ±3% 以内。

## 注意事项
- 考虑税务（卖出可能产生税）
- 考虑交易成本
- 考虑流动性（私募月赎、QDII 限购）
- 大幅再平衡建议分 2-3 次完成
```

## 工具选择建议

### 对个人投资者

**推荐**：
- **Excel / Google Sheets**（最灵活、最简单）
- **Notion 数据库**（结构化好、查询方便）

**不推荐**：
- 复杂的"专业组合管理软件"（除非超过 1000 万）
- 完全依赖券商 APP（不能跨账户合并）

### 自动化建议

如果你懂一点编程，可以：
- 用 Python + 各种 API 自动获取价格
- 用 Google Sheets 的公式自动更新部分价格
- 但要小心 API 不稳定

### 更新频率

- **每日**：不需要（除非全职做投资）
- **每周**：检查重要价格
- **每月**：完整更新组合表
- **每季度**：再平衡评估
- **每年**：完整年度复盘

## 反例 / 风险是什么

**第一，"过度精细化"陷阱。** 不要追求每分钱都精确。月度更新足够。

**第二，"账户合并"麻烦。** 跨账户、跨货币确实麻烦。坚持下来才有价值。

**第三，"目标权重"不是固定的。** 随着年龄、风险偏好、市场环境变化，目标权重也要调整。

**第四，"再平衡"的成本。** 频繁再平衡有交易成本和税务成本。建议季度或半年一次，不是每月。

**第五，"忽视隐藏资产"。** 公积金、社保、退休金账户、保险现金价值都是资产，要纳入整体考虑。

**第六，"短期波动"诱惑。** 看到组合下跌就想"调整结构"是错误。坚持长期目标比追涨杀跌重要。
