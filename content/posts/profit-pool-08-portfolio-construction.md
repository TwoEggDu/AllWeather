## 一、为什么不是把"好股票清单"加起来就完事

第六刀过完，你应该有一份"个股 + 仓位类别"的清单：

- 中际旭创 → 底仓
- 工业富联 → β 底仓
- 新易盛 → 进攻仓
- 海光信息 → 进攻仓（有上限）
- 寒武纪 → 主题仓（有上限）
- ……

这就够了吗？**远远不够**。

把这堆个股加起来，你得到的可能是：

- **相关性过高**：6 只里 4 只都是 AI 算力链，行业 β 一来一起跌
- **回撤不可控**：单只股 -50% 概率小，但 6 只同时 -30% 概率不小
- **没有节奏**：什么时候加仓、什么时候减仓、什么事件触发什么动作，全凭感觉
- **没有对冲**：组合裸跑，宏观风险直接打到净值

**组合不是个股的简单叠加，是有结构、有节奏、有对冲的系统**。

这一刀就是把"个股清单"翻译成"实盘可运作的组合"。

## 二、三层结构：β 仓 + α 仓 + 主题仓

我用的组合骨架很简单——三层：

### β 仓（底仓 / 跟随大盘）
**特征**：业绩档位 ④-⑤、低 α 风险、跟随板块 β
**目标**：稳，不求大涨，靠时间累积
**回撤特征**：单只 -25% ~ -40%
**仓位类型**：30-50% 仓位

### α 仓（进攻 / 边际改善）
**特征**：业绩档位 ③-④、有边际改善逻辑、纯度高
**目标**：吃赛道升级期的"戴维斯双击"
**回撤特征**：单只 -30% ~ -50%
**仓位类型**：20-40% 仓位

### 主题仓（高弹性 / 高风险）
**特征**：业绩档位 ①-②、纯度高、靠想象支撑
**目标**：吃主题情绪/政策预期
**回撤特征**：单只 -50% ~ -80%
**仓位类型**：5-15% 仓位（不超过 20%）

### 三层结构的意义

每一层的"风险/回报"是配对的：
- β 仓波动小，但收益上限也低（年化 15-25%）
- α 仓波动中等，收益弹性中等（年化 30-60%）
- 主题仓波动大，收益弹性也大（年化 -50% 到 +200%）

**组合的整体回撤 = 三层加权回撤**。所以：

> **想要更小的组合回撤 → 提高 β 仓权重**
> **想要更大的弹性 → 提高 α/主题仓权重**

这是显式的取舍，不是凭感觉。

## 三、按目标回撤反推仓位

如果你的组合目标最大回撤 ≤ 25%，倒推三层比例大概是：

| 仓位类型 | 比例 | 单只仓位上限 | 单只最大回撤 |
|---|---|---|---|
| β 仓 | 50% | 15% | -40% |
| α 仓 | 30% | 10% | -50% |
| 主题仓 | 10% | 5% | -70% |
| 现金 | 10% | - | - |

加权预期最大回撤：50% × 40% + 30% × 50% + 10% × 70% = **20% + 15% + 7% = 42%**……

等等，这远超 25% 啊？

是的——**因为这是"三层都同时跌到最大值"的极端假设**。实际中三层的最大回撤不会同时发生（它们的相关性 < 1），所以实际组合最大回撤通常是加权值的 60-70%——也就是 **25-30%** 区间。

如果你的目标是 -25%，再保守一点：**β 仓提到 60%、主题仓减到 5%**。

如果你的目标是 -35%，可以更进攻：**α 仓提到 40%、主题仓加到 15%**。

## 四、跨利润池配置：真正的分散

第六刀讲过：同一利润池的标的相关性极高，**多买不增加分散**。

所以组合层面要做的是**跨利润池配置**：

| 利润池 | 代表标的 | 仓位类别 |
|---|---|---|
| 光模块（AI 算力） | 中际旭创 | β 底仓 |
| 高端 PCB（AI 算力） | 沪电股份 | β 底仓 |
| 液冷（AI 算力） | 英维克 | α 进攻 |
| 机器人零部件 | 汇川技术 | β 底仓 |
| 机器人减速器 | 绿的谐波 | α 进攻 |
| 国产算力 | 海光信息 | α 进攻（上限 8%） |
| AI 应用 | 金山办公 | α 进攻 |
| 国产 GPU | 寒武纪 | 主题（上限 5%） |

**关键观察**：8 个标的覆盖 5 个利润池（AI 算力、机器人零部件、机器人减速器、AI 应用、国产 GPU）。**这种结构在任何单一利润池逻辑松动时都不会全军覆没**。

## 五、节奏管理：事件日历

仓位调整不能凭感觉，要按"事件触发"来。我维护的事件日历大致是这样：

| 事件 | 信号方向 | 仓位调整 |
|---|---|---|
| 北美巨头 capex 上修 | 加仓算力 | β 仓 +10% |
| 北美巨头 capex 下修 | 减仓算力 | α 仓 -1/3 |
| Tesla Optimus 量产推迟 | 减仓机器人 | 主题仓 -1/3 |
| Tesla Optimus 量产兑现 | 加仓机器人 | α 仓 +10% |
| 国产 AI 芯片大单落地 | 加仓主题 | 主题仓 +10% |
| 主要标的财报超预期 ±20% | 个股加仓 | 单股 +5% |
| 主要标的财报不达预期 ±10% | 个股减仓 | 单股 -5% |
| 估值进入历史 90%+ 分位 | 减仓 | 该股 -1/3 |
| 估值进入历史 30%- 分位 | 加仓 | 该股 +1/3 |

**事件日历的真正用途**：减少决策的情绪化。当事件触发时，你照表执行；不触发时，按兵不动。

## 六、对冲：什么时候用、用什么

不是所有时候都需要对冲。三种情形需要：

### 情形 1：宏观系统性风险升高
**信号**：VIX > 25、美债收益率快速变化、地缘冲突
**工具**：股指期货（IH、IF、IC）做空、买入看跌期权
**仓位**：对冲规模 = 30-50% × β 仓

### 情形 2：单一利润池估值过热
**信号**：所在板块估值进入历史 90%+
**工具**：减该板块仓位 + 增加现金
**仓位**：板块仓位 -1/3

### 情形 3：组合裸跑过久
**信号**：连续 6 个月没有触发任何减仓信号
**工具**：增加现金（10-15%）
**仓位**：总仓位 -10%

**不需要对冲的情形**：常规的板块波动、单只股的财务噪音、宏观面无明确恶化。这些靠仓位结构本身就能消化。

## 七、整套实盘流程的"最后一公里"

把前面 7 刀串起来，整个流程是：

```
第一刀（找驱动力）→ 第二刀（拆链）→ 第三刀（识别卡位）
→ 第四刀（打档位）→ 第五刀（回撤归因）
→ 第六刀（个股三筛）→ 第七刀（组合 + 节奏）
                   ↓
              实盘可运作组合
```

每一刀都有显式输出，每一刀的输出是下一刀的输入。**整个流程是机械化、可复用、可校验的**——这就是方法论的价值。

## 八、收尾

完成这一刀，你应该有：

1. **三层组合结构**：β 仓 + α 仓 + 主题仓的具体比例
2. **跨利润池配置**：覆盖 4-6 个不同利润池的个股清单
3. **事件日历**：列出能触发仓位调整的事件清单
4. **对冲规则**：什么情形用什么对冲工具

到此 7 把刀走完——**整套方法论已经能跑实盘了**。

但方法论是抽象的，**真正能让你掌握它的，是亲手用它走一遍**。

接下来两篇是实战推演：

- **实战 1：AI 算力链完整推演**——用前 7 刀完整跑一遍，得出"光模块 + 高端 PCB 重仓"的依据
- **实战 2：机器人链完整推演**——用前 7 刀完整跑一遍，得出"零部件强于本体"的依据

这两篇是把方法论"喂回"具体场景的过程，也是检验你是否真的吸收了前 7 刀的检验题。
