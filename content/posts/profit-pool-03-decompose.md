## 一、教科书拆法为什么不够用

第一刀切完，你已经知道一条链的"驱动力"是谁、有多稳。下一步是看这些钱进入链条之后**怎么流动**。

绝大多数人这一步用的还是教科书拆法：

```
原材料 → 设备 → 制造 → 应用
（上游）  （中游）  （下游）
```

这是按"物料流动方向"拆的——硅片从晶圆厂出来，进入芯片厂，封装成芯片，装到服务器里，卖给云厂。

这种拆法不是错，是**不够用**。

它告诉你"东西怎么走"，没告诉你"价值在哪里被创造和捕获"。一条链上每一个环节都"加工"了上一个环节的产出，但**有的环节加了 50% 的价值，有的环节只加了 5%**——按物理流动看，看不出这种差别。

我自己的拆法不一样，按**价值产生位置**拆，结果通常是 4 层：

```
原料层 → 心脏层 → 承载层 → 消费层
```

听起来差不多，定义完全不同。

## 二、四层是什么意思

### 1. 原料层
**定义**：通用基础物料、设备、工具
**特征**：不直接决定终端产品的核心性能；规模驱动、标准品为主
**例子**（AI 算力链）：晶圆、化学材料、EDA、光刻机、薄膜设备

### 2. 心脏层
**定义**：决定终端产品核心性能的关键模块。IP、专利、技术差距集中的地方
**特征**：替代成本极高、议价权强、毛利通常最厚
**例子**（AI 算力链）：GPU / ASIC 芯片本身

### 3. 承载层
**定义**：把心脏层的能力组合、放大、连接成可用整机/系统
**特征**：技术含量介于心脏和原料之间；常被低估，但**很多链条的真正利润池藏在这里**
**例子**（AI 算力链）：AI 服务器整机、PCB、光模块、HBM、液冷、电源

### 4. 消费层
**定义**：最终用户/买家
**特征**：直接对应"驱动力"——谁付钱
**例子**（AI 算力链）：云厂、互联网巨头、政企客户

**这套分层最重要的价值，是把"心脏层"和"承载层"分开看**。教科书的"中游"是个杂烩——既有芯片也有服务器整机也有 PCB——但这三个东西的利润池位置完全不同。

## 三、为什么这样拆能找到利润池

把"心脏 vs 承载"分开后，几个反直觉的事实就显形：

### 事实 1：心脏层不一定是利润池

GPU 是 AI 算力链的心脏。在国际上，英伟达确实是利润池——毛利率 70%+。
但在国产链上，GPU/ASIC 的国产化还在亏损或微利阶段。**心脏不在国内，国内厂商的"心脏层"暂时不是利润池**。

这就解释了为什么我把寒武纪放在主题仓而不是底仓——心脏没成型，利润池就还没成型。

### 事实 2：承载层经常是被低估的利润池

AI 服务器整机毛利 8-12%，这是承载层"最薄"的部分。但同样在承载层：

| 环节 | 毛利率（典型） |
|---|---|
| 整机 | 8-12% |
| 高端 PCB | 25-30% |
| 光模块 | 25-30% |
| 光器件 | 40%+ |

都是承载层，毛利完全不同。**承载层内部还要再做一次"利润位置"拆解**——同样是把芯片放进机箱，光模块这一段拥有定价权（技术领先 + 客户分散），整机厂没有（同质化竞争 + 客户极度集中）。

### 事实 3：原料层在某些时点也能成为利润池

光刻机被 ASML 一家垄断，毛利极厚。这是原料层的特例——**当原料层有"全球独家"时，议价权可能反过来超过心脏层**。

**总结一句话**：利润池的位置不是固定的，要按每条链的具体格局来定。但**只有先把"心脏 / 承载 / 原料"分清楚，你才有结构去问"利润在哪一层"**。

## 四、案例 1：AI 算力链怎么拆

把第一刀（驱动力）和第二刀（拆链）合起来，AI 算力链长这样：

```
驱动力：北美 4-5 家科技巨头 capex（B 端，高度集中）
   ↓
消费层：Meta / Google / MSFT / AWS / 国内云厂 / 运营商
   ↓
承载层：AI 服务器整机 / 光模块 / PCB / HBM / 液冷 / 电源 / 机箱
   ↓
心脏层：GPU / ASIC（国际：英伟达；国内：海光、寒武、昇腾）
   ↓
原料层：晶圆 / 设备（光刻、刻蚀、薄膜）/ EDA / 材料
```

这样一拆，几个事实就摆出来了：

1. **心脏层国际有利润池（英伟达），国内还没（仍在投入期）** → 国产 GPU 厂商不进底仓
2. **承载层内部分化巨大**：光模块/PCB 有利润池，整机厂没有 → 重仓光模块/PCB，整机厂只配 β 仓
3. **原料层是政策驱动 + 国产替代**，业绩兑现度参差 → 设备龙头（北方华创等）可作平台型配置，但弹性弱于承载层
4. **消费层是"驱动力监控指标"，不是"投资标的"** → 云厂自己也是上市公司，但买它们逻辑不同（看自己业务）

**这张图的实用价值**：第三刀（五问法）开始时，我只在承载层这 6 个环节里挑，不会浪费精力研究消费层。

## 五、案例 2：新能源车链怎么拆

为了证明这套方法不是只适用 AI，再套一次新能源车：

```
驱动力：中欧美月度销量 + 各国补贴政策（C 端 + 政策混合）
   ↓
消费层：终端消费者 + 政策买单
   ↓
承载层：整车厂 / 三电系统 / 智驾系统 / 充电桩
   ↓
心脏层：动力电池 / 电控芯片 / 智驾算法 / 高阶传感器
   ↓
原料层：锂矿 / 镍钴 / 隔膜 / 电解液 / 正负极材料 / 设备
```

应用同样三个判断：

1. **心脏层**：动力电池有利润池（宁德、比亚迪刀片电池），电控/智驾还在博弈期 → 电池龙头是底仓级标的
2. **承载层**：整车厂利润极薄（特斯拉除外）；智驾系统/充电桩是增量赛道但格局未定 → 整车厂不进底仓，承载层选具体细分
3. **原料层**：周期性极强（2021 锂价暴涨、2023 暴跌）→ 不能买在景气顶点，否则估值/业绩双杀

**对比 AI 算力链你会发现**：同样的拆法，结论完全不同——AI 算力链是"承载层重仓 + 心脏国产化暂缓"，新能源车链是"心脏层重仓 + 承载层精选 + 原料周期错配"。

**框架是一样的，结论是动态的**——这就是方法论的价值。

## 六、给一条链画"四层图"的实操步骤

下次你拿到一条新的产业链（医药、军工、机器人都行），按这 4 步走：

1. **先写驱动力**（第一刀的输出）：消费层是谁、付钱属于哪一类
2. **找心脏**：问自己——"这条链的终端产品里，哪一个模块决定了核心性能差距"？那就是心脏。心脏通常 1-2 个，不会很多
3. **铺承载**：心脏之外、消费层之前所有的环节，都是承载层。这一层会很多——5-10 个不奇怪。**承载层是后续拆解的重点**
4. **画原料**：再往上是通用化的设备、材料、工具。原料层除非有"全球独家"否则不重点研究

画完之后，你的笔记本上应该是一张 4 行的图。**这张图就是后面 5 刀的"地图"**。

## 七、收尾

这一刀讲的是"按利润位置拆链"。完成后你应该有：

1. 一张 4 层结构图（消费 / 承载 / 心脏 / 原料）
2. 对每一层"是不是潜在利润池"的初步判断

但这只是初步。**承载层经常有 5-10 个细分环节，到底哪一个是真正的利润池？这要靠下一刀来切**。

下一篇讲**第三刀：五问法识别卡位环节**——给一个 5 个问题的清单，对承载层每个环节问一遍，利润池就显形了。这是从"结构"切到"位置"的关键一刀。
