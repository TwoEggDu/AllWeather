## 1. 缘起

完成五年计划专题（28 篇）之后，整个博客的**理论框架**已经很扎实，但仍存在以下几类缺口：

- **实操执行层缺得最多**：重"为什么"和"配什么"，缺"怎么操作"
- **重要地缘留白**：印度、欧洲、日本、越南等都没专题
- **风险管理深度不足**：尾部对冲、危机剧本、流动性等只散见提及
- **个人财务整体观缺失**：聚焦投资资产，缺房贷、保险、家庭分层
- **进阶主题未覆盖**：超长期债务周期、人口结构、气候投资等

本计划共 26 篇，分 5 个阶段推进，按"价值密度倒序"排列：
**实操执行 → 重要地缘和主题 → 风险管理与个人财务 → 其他地缘 → 进阶专题**

## 4. 文章模板

### 实操执行类模板（阶段 E）

1. **这篇文章要回答的问题**：明确实操痛点
2. **三个/四个选项的对比表**：核心
3. **每个选项的优劣 + 适用人群**
4. **具体操作步骤**（截图描述、关键节点）
5. **常见坑和规避方法**
6. **决策树**：根据你的情况怎么选

### 地缘和主题类模板（阶段 F、H）

1. 这个国家/主题在全球格局中的位置
2. 关键数据和指标
3. 与中国的关系（经贸、地缘、技术）
4. 资产价格表现历史
5. 投资工具和标的
6. 投资风险和反例

### 风险管理与个人财务类模板（阶段 G）

1. 这类风险/问题的真实严重性
2. 工具和方法的对比
3. 具体的执行流程
4. 在不同人生阶段的应用
5. 反例：什么情况下方法失效

### 进阶专题类模板（阶段 I）

1. 框架/概念的核心定义
2. 在中国当前环境下的应用
3. 历史案例验证
4. 投资含义
5. 反例和局限性
