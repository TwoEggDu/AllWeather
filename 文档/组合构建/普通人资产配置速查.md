# 普通人资产配置速查

> 目标：用一份文档回答三个问题：常见大类资产是什么、不同宏观环境下谁更占优、普通人如何把这些判断落成可执行的配置框架。

---

## 这份文档解决什么问题

很多人接触资产配置时，会很快掉进两个坑：

- 只记住了 `股、债、商、金` 这些名词，但不知道它们为什么涨跌
- 知道一点宏观判断，却不知道该如何转成实际动作

这份文档只做最小可用版本：

- 先建立大类资产地图
- 再用 `增长 x 通胀` 四象限理解不同资产的优势环境
- 最后给出普通人能执行的配置框架和月度宏观仪表盘

它不是预测手册，也不是选基推荐，而是一张能反复查看的工作底图。

---

## 一、大类资产最小地图

### 1. 股票

- 本质：买公司的所有权和未来盈利
- 主要收益来源：盈利增长、估值提升、分红
- 通常更受益于：经济扩张、盈利改善、流动性不太紧的时候
- 主要风险：衰退、加息、估值过高后的回撤

### 2. 债券

- 本质：把钱借给政府或企业，赚票息和价格波动
- 主要收益来源：利息、利率下行带来的债券价格上涨
- 通常更受益于：经济走弱、通胀回落、降息预期升温的时候
- 主要风险：高通胀、加息、信用恶化

### 3. 商品

- 本质：对能源、金属、农产品等实物资源的价格暴露
- 主要收益来源：价格上涨
- 通常更受益于：通胀上行、供给受限、需求仍强的时候
- 主要风险：需求塌陷、美元走强、库存压力

### 4. 黄金

- 本质：不产生现金流，更像对货币购买力、信用风险和避险需求的对冲
- 主要收益来源：价格上涨
- 通常更受益于：避险升温、实际利率回落、滞胀担忧上升的时候
- 主要风险：实际利率高企、美元走强、风险偏好过强

### 5. 现金与短债

- 本质：流动性缓冲
- 主要收益来源：存款利息、货基或短债收益
- 通常更受益于：高利率、高不确定性、高波动环境
- 主要风险：长期跑不赢通胀

### 6. 其他常见资产

- `房地产 / REITs`：更依赖租金、融资成本和经济稳定性
- `外汇`：更像宏观交易工具，不是大多数普通人的长期核心收益来源
- `另类资产`：私募、对冲、基础设施、艺术品等，门槛高、流动性差
- `加密资产`：高波动、高争议，通常更接近另类资产而不是核心配置

## 二、四种宏观环境下，谁通常更占优

最常用的框架只有两个变量：

- `增长`：走强还是走弱
- `通胀`：上行还是下行

把它们交叉起来，常见结论如下：

| 宏观环境 | 通常更占优 | 核心逻辑 |
|---|---|---|
| `增长上行 + 通胀温和` | `股票` | 盈利改善，风险偏好上升 |
| `增长走弱 + 通胀回落` | `债券` | 利率见顶或下行预期增强 |
| `增长上行 + 通胀上行` | `商品` | 需求强、价格涨、供需弹性大 |
| `增长走弱 + 通胀仍高` | `黄金`，其次部分商品 | 滞胀和避险担忧上升，黄金相对更有优势 |

可以直接记成一句话：

- `看增长`：强就偏股
- `看降息`：近就偏债
- `看通胀`：升就看商品
- `看滞胀和避险`：重就看黄金和现金

要注意，这不是机械公式。短中期表现还会受到 `估值`、`政策力度`、`市场是否提前定价` 的影响。

## 三、为什么“股债商金”能分散风险

核心原因不是“它们一定反着走”，而是：

> 它们赚钱的来源不同，最敏感的宏观变量也不同。

- `股票`更敏感于：盈利、估值、风险偏好
- `债券`更敏感于：利率、通胀、信用
- `商品`更敏感于：供需、库存、地缘冲击
- `黄金`更敏感于：实际利率、美元、避险和信用担忧

所以当一种宏观情景对某类资产不利时，另一类资产可能部分对冲：

- 衰退时，股票通常承压，优质债券更可能发挥防守作用
- 通胀上行时，债券常承压，商品和黄金更可能受益
- 避险升温时，黄金和现金往往比高波动风险资产更稳

分散风险的本质是：

- 分散 `风险来源`
- 不是简单分散 `资产数量`

同样需要记住：

- 分散能降低风险，不等于消灭亏损
- 有些阶段股债会一起跌，尤其是在 `高通胀 + 快速加息` 时
- 商品和黄金本身波动也不低，不能把“能对冲”理解成“适合重仓”

## 四、普通人怎么做资产配置

### 1. 先按用途分层

先不要急着定比例，先回答这笔钱什么时候要用：

- `应急金`：先留出 `3 到 6 个月生活费`，放在现金类资产里
- `3 年内要用的钱`：以现金、短债为主，不要靠股票承担刚性支出
- `3 到 7 年的钱`：可以用股债平衡
- `7 年以上不用的钱`：股票可以作为核心仓位，债券负责降波动

### 2. 再选一个基准配置

下面这些比例，默认是指长期投资资金，不含应急金：

| 类型 | 参考配置 | 适用场景 |
|---|---|---|
| `保守型` | `30% 股 / 70% 债` | 更看重波动控制和资金稳定性 |
| `平衡型` | `60% 股 / 40% 债` | 兼顾增长和回撤承受 |
| `进取型` | `80% 股 / 20% 债` | 资金长期不用，能承受较大波动 |

### 3. 普通人的执行优先级

- 股票部分：优先宽基、低费率、分散化工具
- 债券部分：优先高信用、流动性好的债券工具
- 黄金、商品、REITs：可以做小比例补充，但不建议替代股债核心仓

### 4. 再平衡规则

- 每 `6 到 12 个月` 做一次常规再平衡
- 或者当某类资产明显偏离目标权重时再调
- 宏观判断只做 `10% 到 15%` 的倾斜，不要把长期仓位全切来切去

一句话版：

> 钱越快要用，越偏现金和债；钱越久不用，越可以偏股票。

## 五、宏观信号 -> 资产动作

普通人最实用的做法，不是追求预测精度，而是看到信号后知道组合该往哪边微调。

| 宏观信号 | 常见含义 | 资产动作 |
|---|---|---|
| `PMI 回升、盈利上修、就业稳` | 增长改善 | `偏股`，少一点长债 |
| `PMI 走弱、失业上升、盈利下修` | 增长走弱 | `加债`，降低股票风险暴露 |
| `CPI / PPI 持续回落` | 通胀降温 | `利好债`，成长股也更受益 |
| `通胀重新上行、油价大涨` | 通胀压力回升 | `减长债`，适度看商品 / 黄金 |
| `央行停止加息或准备降息` | 流动性边际转松 | `加债`，优质股票也受益 |
| `央行持续收紧、实际利率上行` | 估值受压 | `减长久期债`，股票更偏防御 |
| `信用利差明显走阔` | 风险偏好下降 | `减高风险资产`，加高等级债 / 现金 |
| `避险情绪升温` | 不确定性上升 | `加黄金、国债、现金` |

执行时记住三条规则：

- 不因为一条单月数据就大调仓
- 尽量看 `连续 2 到 3 个月` 的方向
- 宏观信号用于 `微调`，不是替代长期配置框架

## 六、普通人版宏观信号仪表盘

下面这个版本，目标是 `每月看一次` 就够用。

| 指标 | 看什么 | 绿灯 | 黄灯 | 红灯 | 资产倾向 |
|---|---|---|---|---|---|
| `增长` | `PMI` | `>50` 且连续回升 | 在 `50` 附近震荡 | `<50` 且走弱 | 绿偏 `股`，红偏 `债` |
| `通胀` | `CPI / PPI + 油价` | 通胀回落、油价平稳 | 变化不大 | 通胀再起、油价上冲 | 绿偏 `债`，红偏 `商 / 金` |
| `流动性` | `央行态度 / 政策利率` | 停止加息或转松 | 中性 | 持续收紧 | 绿偏 `股 / 债`，红偏 `现金` |
| `长端利率` | `10 年期国债收益率` | 连续下行 | 横盘 | 连续上行 | 绿偏 `债`，红减 `长债` |
| `信用压力` | `信用利差 / 违约担忧` | 利差收窄 | 小幅走阔 | 明显走阔 | 绿偏 `股`，红偏 `高等级债 / 现金` |
| `避险压力` | `美元、波动率、地缘风险` | 风险平稳 | 一般 | 避险升温 | 红偏 `黄金 / 现金` |

### 怎么读结果

- `增长绿 + 通胀黄/绿 + 流动性绿`：偏 `股票`
- `增长红 + 通胀绿 + 利率下行`：偏 `债券`
- `增长黄/绿 + 通胀红 + 利率上行`：偏 `商品`
- `增长红 + 通胀红 + 避险红`：偏 `黄金 / 现金`

### 一页模板

```text
本月宏观仪表盘

1. 增长（PMI）：
2. 通胀（CPI / PPI / 油价）：
3. 流动性（央行）：
4. 长端利率（10Y 国债）：
5. 信用压力：
6. 避险压力：

结论：
- 当前增长：上行 / 下行
- 当前通胀：上行 / 下行
- 组合倾向：偏股 / 偏债 / 偏商 / 偏金现金
- 本月动作：只做 10% 到 15% 的微调
```

## 八、最后记住三句话

- 没有任何一种资产能在所有环境里长期都赢
- 真正的分散，是分散不同的风险来源
- 对普通人来说，长期基准配置比短期宏观判断更重要
